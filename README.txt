# jQuery mCalc - Mortgage Calculator

This is a fork of the jquery-mcalc plugin. It has been updated to run on jQuery 1.4.2 and jQuery UI 1.8.4

Usage is identical to the original (with the exception of including the updated jQuery code). To view original documentation visit:

http://code.google.com/p/jquery-mcalc/
